Requerimientos Funcionales
El sistema debe consultar al usuario sus datos de cuenta para poder entrar a la aplicación.
El sistema debe tener disponible una opción de revisar las rutinas de un videojuego si el usuario entra a revisar los detalles de éste. 
El sistema debe tener a disponibilidad un acceso rápido de menús.
El sistema debe detallar datos del juego tales como: nombre, una descripción corta, el modo de jugabilidad y la cantidad de rutinas pertenecientes al juego.
El sistema debe permitir al usuario editar: su lista de juegos, su lista de rutinas, su lista de amigos. 
	También debe permitir editar su perfil de usuario
El sistema debe especificar el objetivo principal de la rutina, objetivos a corto plazo y el porcentaje de completado de la rutina. 

----------------------------------------------------------------
Requerimientos no Funcionales
El sistema debe resguardar y almacenar de forma segura cualquier dato que ingrese el usuario
El sistema debe darle la opción al usuario si desea compartir alguno de estos datos con terceros.
El sistema no debe durar demasiado tiempo en almacenar y/o realizar alguna acción.
